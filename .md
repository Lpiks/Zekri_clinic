# 🦷 Zekri Dental Clinic - MVP Architecture & System Prompt

## 📋 1. Project Context

**Project Name:** Zekri Dental Clinic - Premium Frontend MVP
**Goal:** Create an ultra-premium, high-conversion frontend prototype for a high-end dental clinic specializing in Zirconia crowns and aesthetic dentistry.
**Vibe/Aesthetic:** "Dark Luxury", Swiss Clinic, Legacy, Trust. High-end visual experience.

## 🛠 2. Tech Stack

- **Framework:** React + Vite
- **Routing:** React Router v6
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion (Crucial for the "Premium" feel)
- **Icons:** Lucide React
- **State Management (MVP):** React Context API + `localStorage` (to simulate DB for the Admin side)

## 🎨 3. Design System (Dark Luxury)

- **Backgrounds:** `#0A0B10` (Obsidian Black) and `#11131A` (Soft Dark)
- **Accents (Gold/Brass):** `#D4AF37` (Primary), `#C5A059` (Hover states)
- **Text:** `#F9FAFB` (Primary Text), `#9CA3AF` (Secondary Text)
- **Typography:** \* Headings (`h1`, `h2`): `Playfair Display` or `Cormorant Garamond`
  - Body text: `Plus Jakarta Sans` or `Inter`

**Task:** We are building a Minimum Viable Product (MVP) for "Zekri Dental Clinic". The focus is on creating a "Dark Luxury", high-end aesthetic. You will act as my pair programmer.

**Rules & Guidelines:**

1. **Component Driven:** Always break down large pages into small, reusable components.
2. **Styling:** Use Tailwind CSS exclusively. Use the specific color palette: Obsidian background (`#0A0B10`), Gold accents (`#D4AF37`), and white/gray text.
3. **Animations:** Do not build static pages. Every page load should have a smooth `Framer Motion` transition (e.g., `initial={{ opacity: 0, y: 20 }}`, `animate={{ opacity: 1, y: 0 }}`, `transition={{ duration: 0.6, ease: "easeOut" }}`).
